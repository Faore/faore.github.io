## Work-Related Projects
* **[JuliaFischer.com](https://www.juliafischer.com)** - Website and subscription service for violinist Julia Fischer. 
Written using Ruby, Sinatra, SASS, jQuery and Bootstrap 3.
* **ResNet Tools** - A web application written in PHP and [Laravel](http://www.laravel.com) for managing copyright 
complaint tickets, sending text messages to clients, automatically annotating tickets, and managing lab computers, 
their local accounts, and storing audit information.
* **Lab Computer Service** - A Windows service designed to securely store audit information, manage the guest accounts 
and sync with a central server over the network. Written in C#.
* **ResNet Alerts Server** - A web application for managing communications about network status and events for a mobile 
application.
* **ResNet Alerts** - A mobile application prototype for communicating network status and events, as well as letting 
users find help desk hours and submitting help desk tickets from their phone.

maps.
